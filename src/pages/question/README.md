## webstorm
### Vue项目使用路径别名没有提示信息
- `windows` File > Settings > Languages and Frameworks >JavaScript | Webpack
- `macOS`  WebStorm | Preferences | Languages and Frameworks | JavaScript | Webpack
- 选择配置文件路径为 `node_modules/@vue/cli-service/webpack.config.js`即可。
